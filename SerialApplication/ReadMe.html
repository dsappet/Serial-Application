About: Initially this program was designed to perform a more specific task with the serial port for testing specific hardware. I have tried to make it more general purpose and make it configurable.

Written in C# the program utilizes the system.IO functions for access to the serial port. The listener function subscribes to an event fired when data is received on the port. The program has an option to send a reply string every time data is received on the port. The write button can write just once, usually used to write then wait for a reply. The looping functionality writes the string in the input box.
